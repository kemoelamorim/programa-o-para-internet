O que é a internet ? O que é a web? São a mesma coisa?

  • Internet:
    •             É uma rede de conexão de computadores que permite a comunicação e o compartilhamento de dados e informações,
            Sua funcionalidade acontence por meio de protocolos, o mais importante é o TPC/IP
  • Web:
    •             Weab é uma aplicação que roda na internet, 
            A web uso o protocolo http/https para promover essa transferência de informações
            e depende de browsers(navegadores) para apresensar as informações

    O que é HTTP?

            É um acronimo para "HyperText Transfer Protocol", 
        que significa "Protocolo de Transferencia de Texto"
        este protocolo estabelece a comunicação de dados da World Wide Web

    O que é HTML?

            HTML significa "Linguagem de marcação de HiperTexto"
        É uma linguagem de Marcação para a construção de páginas na web

    Cite e explique o que são tags HTML. 

            Tags HTML são as estuturas que organizam as informações em uma página na Web,
        cada tag tem uma semântica que da um significado para o navegador apresentar a informação
        As tag podem tem abertura e fechamento, ou apenas abertura. Também podem receber propriedades.
